# 🤖 AI-Powered E-Whiteboard

An AI-powered virtual whiteboard built using Python, OpenCV, MediaPipe, and Tesseract OCR. The system allows users to draw using hand gestures, automatically correct shapes, recognize handwritten text, and export notes as PDF.

---

## ✨ Features

- ✋ Hand gesture-based drawing
- 🎯 Real-time hand tracking using MediaPipe
- 📝 Handwritten text recognition using Tesseract OCR
- 🔷 Automatic shape correction
- 🎨 Virtual drawing canvas
- 🧹 Eraser functionality
- 📄 Export notes as PDF
- ⚡ Real-time processing using OpenCV
